Se connecter a redstonia

Redstonia est un serveur minecraft 1.19.2. L'ip est Redstonia.tk (provisoire). Les crack sont autorisé. Voici la liste des modes

Comment jouer sur Redstonia ?

Pour commencer il faut avoir une version de forge supporté.
Demarrer le logiciel "Exécuter" puis mettez "%appdata%" et appuiez sur ok.
Allez sur le dossier ".minecraft" puis glissez create dans le dossier "mods".
Demarrer le lanceur minecraft sur la version de forge.
Une fois sur Minecraft allez sur "Multiplayer", "Direct Connexion" mettez l'ip et appuiez sur "Join Server"
